# Usage
:::Global Import
```ts
import { I18n, t } from 'abm-ui';
```
:::On-demand Import
```ts
import { I18n, t } from 'abm-ui/component/i18n';
```
:::Registration Import
```ts
import 'abm-ui/component/i18n';
```
:::

```tsx
t('ui.ok') // Recommended
<I18n>ui.confirm</I18n>
<I18n key="ui.cancel"/>
```

```html
<abm-i18n>ui.confirm</abm-i18n>
<abm-i18n key="ui.cancel"></abm-i18n>
```

# Properties

## `key`
String type, translation key.

## `args`
[`LocaleArgs`](../utils/locale#type-localeargs) type, translation parameters.

# Methods

## `update`
No parameters, no return value, updates translation content.

# Function `t`
Creates and returns an I18n component instance.

Parameters:
- `key`: Translation key
- `args`: Optional, translation parameters

Return value: `I18n` instance
