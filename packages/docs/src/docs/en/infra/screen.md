# Object `safeRect`
Screen safe area settings.

## Properties

| Property Name | Description |
| ------------- | ----------- |
| `top`         | Top safe distance |
| `right`       | Right safe distance |
| `bottom`      | Bottom safe distance |
| `left`        | Left safe distance |

## Read-only Properties

| Property Name | Description |
| ------------- | ----------- |
| `vStart`      | Vertical safe area start coordinate |
| `vEnd`        | Vertical safe area end coordinate |
| `vSize`       | Vertical safe area size |
| `hStart`      | Horizontal safe area start coordinate |
| `hEnd`        | Horizontal safe area end coordinate |
| `hSize`       | Horizontal safe area size |
