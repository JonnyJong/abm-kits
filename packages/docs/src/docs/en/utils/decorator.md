# Decorator `typeCheck`
Used for static, instance, or accessor methods, checks if parameter types meet requirements.
Refer to the [`verifyType`](./type#function-verifytype) function for checking methods.

# Decorator `toType`
Used for static, instance, or accessor methods, converts parameters to the specified type.
