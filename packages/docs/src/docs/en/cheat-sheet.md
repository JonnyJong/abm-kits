# UI Layers

| `z-index` | CSS Class                    | Description     |
| --------- | ---------------------------- | --------------- |
| 50        | `.overlay`                   | Overlay         |
|           | `.backdrop`、`.backdrop-dim` | Overlay backdrop |
| 80        | `.ui-toast-container`        | Toast container |
| 90        | `.ui-nav`                    | Navigation indicator |
| 100       | `.ui-tooltip`                | Tooltip         |
