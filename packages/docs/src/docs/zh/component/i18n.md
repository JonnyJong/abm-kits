# 使用
:::全局导入
```ts
import { I18n, t } from 'abm-ui';
```
:::按需导入
```ts
import { I18n, t } from 'abm-ui/component/i18n';
```
:::注册导入
```ts
import 'abm-ui/component/i18n';
```
:::

```tsx
t('ui.ok') // 推荐
<I18n>ui.confirm</I18n>
<I18n key="ui.cancel"/>
```

```html
<abm-i18n>ui.confirm</abm-i18n>
<abm-i18n key="ui.cancel"></abm-i18n>
```

# 属性

## `key`
字符串类型，翻译键名。

## `args`
[`LocaleArgs`](../utils/locale#类型-localeargs) 类型，翻译参数。

# 方法

## `update`
无参数无返回值，更新翻译内容。

# 函数 `t`
创建并返回 I18n 组件实例。

参数：
- `key`：翻译键名
- `args`：可选，翻译参数

返回值：`I18n` 实例
