| 类名             | 描述                               | 示例                                                                                                                   |
| ---------------- | ---------------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| `.group`         | 将内部多个 UI 控件组合在一起       | <div class="group"><abm-btn>Start</abm-btn><abm-text-box>Write something...</abm-text-box><abm-btn>End</abm-btn></div> |
| `.overlay`       | 覆盖层                             |                                                                                                                        |
| `.backdrop`      | 覆盖层背景（透明）                 |                                                                                                                        |
| `.backdrop-dim`  | 覆盖层背景（半透明）               |                                                                                                                        |
| `.surface`       | 基础表面容器，带内边距、边框和圆角 | <div class="surface"></div>                                                                                            |
| `.surface-glass` | 毛玻璃效果表面                     | <div class="surface-glass"></div>                                                                                      |
| `.text-display`  | 大标题样式                         | <div class="text-display">Text Display</div>                                                                           |
| `.text-title`    | 标题样式                           | <div class="text-title">Text Title</div>                                                                               |
| `.text-subtitle` | 副标题样式                         | <div class="text-subtitle">Text Subtitle</div>                                                                         |
| `.safe-size`     | 安全尺寸                           |                                                                                                                        |
| `.safe-inset`    | 安全位置                           |                                                                                                                        |
