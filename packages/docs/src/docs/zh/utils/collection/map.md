# 类 `TwoKeyMap`
二维映射表，使用两个键（`K1`, `K2`）来索引值（`V`）。相当于 `Map<K1, Map<K2, V>>` 的封装，提供更方便的操作接口。

## 方法
### `has(k1, k2)`
检查是否存在由两个键指定的值。
- 参数：
  - `k1`：第一维键
  - `k2`：第二维键
- 返回值：如果两个键对应的值存在则返回 `true`，否则返回 `false`

### `get(k1, k2)`
根据两个键获取对应的值。
- 参数：
  - `k1`：第一维键
  - `k2`：第二维键
- 返回值：对应的值，如果不存在则返回 `undefined`

### `set(k1, k2, value)`
设置指定键对应的值。
- 参数：
  - `k1`：第一维键
  - `k2`：第二维键
  - `value`：要设置的值
- 返回值：返回当前实例，支持链式调用

### `delete(k1, k2)`
删除由两个键指定的值。
- 参数：
  - `k1`：第一维键
  - `k2`：第二维键
- 返回值：如果成功删除返回 `true`，如果键不存在则返回 `false`

### `hasInnerMap(k1)`
检查是否存在指定第一维键对应的内部 Map。
- 参数：`k1`：第一维键
- 返回值：如果存在对应内部 Map 则返回 `true`，否则返回 `false`

### `getInnerMap(k1)`
获取第一维键对应的内部 Map（包含所有第二维键值对）。
- 参数：`k1`：第一维键
- 返回值：对应的内部 Map，如果不存在则返回 `undefined`

### `setInnerMap(k1, innerMap)`
设置第一维键对应的内部 Map。
- 参数：
  - `k1`：第一维键
  - `innerMap`：要设置的内部 Map
- 返回值：返回当前实例，支持链式调用
- 异常：如果传入的参数不是 Map 实例，则抛出 `TypeError`

### `deleteInnerMap(k1)`
删除第一维键对应的整个内部 Map。
- 参数：`k1`：第一维键
- 返回值：如果成功删除返回 `true`，如果键不存在则返回 `false`

### `entries()`
返回外层映射的默认迭代器，遍历第一维键及其对应的内部 Map。
- 返回值：第一维键及其对应的内部 Map 的迭代器

### `innerEntries()`
遍历所有内部映射的键值对，生成第二维键和值的迭代器。
- 返回值：第二维键和值的迭代器

### `allEntries()`
遍历所有条目，生成完整的三元组迭代器。
- 返回值：完整的三元组（k1, k2, v）的迭代器

### `keys1()`
返回所有第一维键的迭代器。
- 返回值：第一维键的迭代器

### `keys2()`
遍历所有第二维键，生成其迭代器。
- 返回值：第二维键的迭代器

### `values()`
遍历所有存储的值，生成其迭代器。
- 返回值：值的迭代器

# 类 `SetMap`
多值映射集合，将键映射到值的集合。每个键对应一个不重复的值集合，适用于一对多关系的数据结构。

## 属性
- `size`：获取当前映射中键的数量

## 方法
### `hasKey(key)`
检查是否包含指定的键。
- 参数：`key` - 要检查的键
- 返回值：如果包含该键则返回 `true`，否则返回 `false`

### `has(key, value)`
检查是否包含指定的键值对。
- 参数：
  - `key` - 要检查的键
  - `value` - 要检查的值
- 返回值：如果包含该键值对则返回 `true`，否则返回 `false`

### `get(key)`
获取指定键对应的值集合。
- 参数：`key` - 要查询的键
- 返回值：对应的值集合，如果键不存在则返回 `undefined`

### `getOne(key)`
获取指定键对应的值集合中的任意一个值。
- 参数：`key` - 要查询的键
- 返回值：对应的值集合中的任意一个值，如果键不存在或值集合为空则返回 `undefined`

### `set(key, values)`
设置指定键对应的值集合（替换现有值）。
- 参数：
  - `key` - 要设置的键
  - `values` - 要设置的值集合（可迭代对象）
- 返回值：当前 SetMap 实例（支持链式调用）

### `add(key, value)`
向指定键对应的值集合中添加一个值。
- 参数：
  - `key` - 要添加值的键
  - `value` - 要添加的值
- 返回值：当前 SetMap 实例（支持链式调用）

### `delete(key, value)`
从指定键对应的值集合中删除一个值。
- 参数：
  - `key` - 要删除值的键
  - `value` - 要删除的值
- 返回值：当前 SetMap 实例（支持链式调用）

### `deleteKey(key)`
删除整个键及其对应的值集合。
- 参数：`key` - 要删除的键
- 返回值：当前 SetMap 实例（支持链式调用）

### `clear()`
清空所有映射关系。
- 返回值：当前 SetMap 实例（支持链式调用）

### `entries()`
获取键值对的迭代器。
- 返回值：返回键值对的迭代器，其中每个值是 Set<V>

### `keys()`
获取所有键的迭代器。
- 返回值：返回键的迭代器

### `values()`
获取所有值集合的迭代器。
- 返回值：返回值集合的迭代器

# 类 `BiMap`
双向映射图，支持从左键到右值和从右值到左键的双向查找。

## 属性
- `size`：获取键值对的数量
- `inverse`：获取反向视图，类型为 `BiMap<R, L>`

## 方法
### `clear()`
清空所有键值对。

### `delete(left)`
通过左键删除对应的键值对。
- 参数：`left` - 要删除的左键
- 返回值：如果存在并删除了该键值对则返回 `true`，否则返回 `false`

### `forEach(callbackfn, thisArg?)`
遍历 BiMap 中的所有键值对。
- 参数：
  - `callbackfn` - 为每个键值对调用的函数，接收三个参数：右值、左键和 BiMap 本身
  - `thisArg` - 执行 callbackfn 时使用的 this 值（可选）

### `get(left)`
通过左键获取对应的右值。
- 参数：`left` - 要查找的左键
- 返回值：如果存在则返回对应的右值，否则返回 `undefined`

### `has(left)`
检查 BiMap 中是否存在指定的左键。
- 参数：`left` - 要检查的左键
- 返回值：如果存在则返回 `true`，否则返回 `false`

### `set(left, right)`
设置或更新一个键值对。
- 参数：
  - `left` - 左键
  - `right` - 右值
- 返回值：返回 BiMap 实例本身，支持链式调用

### `entries()`
返回一个包含所有 [左键, 右值] 对的迭代器。
- 返回值：[左键, 右值] 对的迭代器

### `keys()`
返回一个包含所有左键的迭代器。
- 返回值：左键的迭代器

### `values()`
返回一个包含所有右值的迭代器。
- 返回值：右值的迭代器
