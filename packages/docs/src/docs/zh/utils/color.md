# 类型 `RGB`
数字三元组类型，各通道名称及范围如下：
- R: 0~255
- G: 0~255
- B: 0~255

# 类型 `RGBA`
数字四元组类型，各通道名称及范围如下：
- R: 0~255
- G: 0~255
- B: 0~255
- A: 0~255

# 类型 `HSL`
数字三元组类型，各通道名称及范围如下：
- H: 0~360
- S: 0~1
- L: 0~1

# 类型 `HSLA`
数字四元组类型，各通道名称及范围如下：
- H: 0~360
- S: 0~1
- L: 0~1
- A: 0~1

# 类型 `Oklch`
数字三元组类型，各通道名称及范围如下：
- L: 0~1
- C: 0~Infinity
- H: 0~360

# 类型 `OklchAlpha`
数字四元组类型，各通道名称及范围如下：
- L: 0~1
- C: 0~Infinity
- H: 0~360
- A: 0~1

# 函数 `rgb2hsl`
将 RGB 值转换为 HSL 值。

# 函数 `hsl2rgb`
将 HSL 值转换为 RGB 值。

# 函数 `rgb2oklch`
将 RGB 值转换为 Oklch 值。

# 函数 `oklch2rgb`
将 Oklch 值转换为 RGB 值。

# 类 `Color`

## 方法

### 设置/获取值
以下所有方法无参数传入时为获取值，有参数传入时为设置值。
| 方法名       | 说明                              |
| ------------ | --------------------------------- |
| `rgb`        | 设置/获取 RGB 值                  |
| `rgba`       | 设置/获取 RGBA 值                 |
| `hex`        | 设置/获取 16 进制值               |
| `hexa`       | 设置/获取 16 进制值（包含 alpha） |
| `hsl`        | 设置/获取 HSL 值                  |
| `hsla`       | 设置/获取 HSLA 值                 |
| `oklch`      | 设置/获取 Oklch 值                |
| `oklchAlpha` | 设置/获取 OklchAlpha 值           |
| `alpha`      | 设置/获取 alpha 值                |
| `alphaByte`  | 设置/获取 alpha 字节值            |

### `clone`
克隆颜色实例。

### `invert`
创建与当前颜色值相反的颜色实例。

### `toString`
将颜色实例转换为 HEX 格式字符串。

### `parseHEX`
将可能为 HEX 格式的字符串转换为 RGBA 值。

以下为各种输入的转换和输出结果：
- 完整的 HEX 格式：
  - `abc` -> `#abc` -> `#aabbcc` -> `[170, 187, 204, 255]`
  - `abcd` -> `#abcd` -> `#aabbccdd` -> `[170, 187, 204, 221]`
- 残缺的 HEX 格式：
  - `a` -> `aaa` -> `[170, 170, 170, 255]`
  - `ab` -> `aba` -> `[170, 187, 170, 255]`
  - `abcde` -> `#abcdea` -> `[171, 205, 234, 255]`
  - `abcdef3` -> `#abcdef33` -> `[171, 205, 239, 51]`
- 过长的 HEX 格式：
  - `0123456789abcdef` -> `#01234567` -> `[1, 35, 69, 103]`
- 不支持的 HEX 格式：
  - `` -> `null`
  - `#` -> `null`
