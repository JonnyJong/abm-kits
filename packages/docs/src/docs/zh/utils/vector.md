# 类型 `Rect`
通过矩形四边坐标声明一个矩形。

# 类型 `Direction4`
四向方向枚举。

# 类型 `Direction8`
八向方向枚举。

# 类型 `Vec2`
数字二元组，二维向量。

# 对象 `Vector2`
二维向量运算函数集。

| 函数名             | 描述                                                                |
| ------------------ | ------------------------------------------------------------------- |
| `isVec2`           | 判断是否为二维向量                                                  |
| `toVec2`           | 将值转换为二维向量                                                  |
| `zero`             | 创建零向量                                                          |
| `isZero`           | 判断是否为零向量                                                    |
| `radians`          | 计算向量的弧度或两坐标连线弧度                                      |
| `length`           | 计算向量的长度或两坐标距离                                          |
| `add`              | 向量加法                                                            |
| `sub`              | 向量减法                                                            |
| `mul`              | 计算向量与标量的乘积或与另一向量的向量积                            |
| `dot`              | 计算向量与标量的乘积或与另一向量的数量积/点积                       |
| `scale`            | 两向量逐分量乘                                                      |
| `clamp`            | 将目标向量限制在指定范围内                                          |
| `scalarProjection` | 计算向量到另一向量的标量投影                                        |
| `normalize`        | 归一化向量，将向量长度设为 1，方向不变；若向量长度为 0 则返回零向量 |
