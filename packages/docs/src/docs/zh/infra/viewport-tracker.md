# 对象 `viewportTracker`

## 函数 `lock`
将视口追踪目标设置为指定元素，确保目标元素在视口内可见。当目标元素从文档树移除时，自动移除追踪目标。

## 函数 `unlock`
移除当前的视口追踪目标，只有输入的目标元素与当前追踪目标相同时，才会移除。

## 函数 `on`、`off`
添加或移除视口追踪事件监听器。

| 事件名     | 参数列表 | 描述                                   |
| ---------- | -------- | -------------------------------------- |
| `scrolled` | `()`     | 为使目标元素可见，使视口发生滚动时触发 |
