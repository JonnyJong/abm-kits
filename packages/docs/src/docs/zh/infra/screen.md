# 对象 `safeRect`
屏幕安全区设置。

## 属性

| 属性名   | 描述         |
| -------- | ------------ |
| `top`    | 顶部安全距离 |
| `right`  | 右侧安全距离 |
| `bottom` | 底部安全距离 |
| `left`   | 左侧安全距离 |

## 只读属性

| 属性名   | 描述               |
| -------- | ------------------ |
| `vStart` | 垂直安全区起始坐标 |
| `vEnd`   | 垂直安全区结束坐标 |
| `vSize`  | 垂直安全区大小     |
| `hStart` | 水平安全区起始坐标 |
| `hEnd`   | 水平安全区结束坐标 |
| `hSize`  | 水平安全区大小     |
