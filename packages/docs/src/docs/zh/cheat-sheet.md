# UI 层级

| `z-index` | CSS 类名                     | 描述         |
| --------- | ---------------------------- | ------------ |
| 50        | `.overlay`                   | 覆盖层       |
|           | `.backdrop`、`.backdrop-dim` | 覆盖层背景   |
| 80        | `.ui-toast-container`        | 吐司通知容器 |
| 90        | `.ui-nav`                    | 导航指示器   |
| 100       | `.ui-tooltip`                | 工具提示     |
